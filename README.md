# ЗВЕЗДНЫЙ КОДЕКС: НАСЛЕДИЕ ЦИВИЛИЗАЦИЙ

## ОБЗОР ПРОЕКТА

**«Звездный Кодекс»** - научно-фантастическая RPG об исследовании космоса, где интеллект - главное оружие. Изучайте древние цивилизации, заполняйте космическую энциклопедию и решайте научные головоломки.

## КОНЦЕПЦИЯ

### СЕТТИНГ

Галактика хранит следы исчезнувших цивилизаций. Вы - археолог-первооткрыватель, исследующий заброшенные миры через древнюю Сеть connecting worlds.

**Цели:**
* Исследовать процедурно генерируемые планетные системы
* Сканировать артефакты и формы жизни для Кодекса
* Решать научные головоломки для доступа к новым зонам
* Восстановить историю павших цивилизаций

## ОСОБЕННОСТИ ГЕЙМПЛЕЯ

* Свободное исследование космоса с разными типами планет
* Образовательный контент, органично встроенный в геймплей
* Нелинейное прохождение с разными путями развития
